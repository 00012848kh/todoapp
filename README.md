# TO DO APPLICATION

This app allows ti track the to dos.

#### Instructions
1.Download the source code
```bash 
git clone https://github.com/00012848kh/todoapp.git
```

or just download the *.zip file

2.Install deps
```bash
npm install
```

3.Run the app
```bash
node app
```

#### Deps list
- express
- pug
